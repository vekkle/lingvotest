Данный файл содержит перечень доступных в этом репозитории файлов.

readme — данный файл.

words_selected — файл, содержащий список из 360 слов, которые были отобраны из общего словника ЛЭС на этапе первичного отбора.
criteria_for_words_selection — перечень критериев, по которым отбраковывались слова из общего словника ЛЭС (1399 словарных входов).
criteria_thematic_groupping — перечень тематических категорий, на которые делились слова из списка первично отобранных слов.

words_final — итоговой перечень из 120 слов, которые вошли в пилотное исследование.

lingvotest_rawdata — свежая выгрузка данных из сервиса "Google Формы" без дополнительной обработки.
lingvotest_data — данные пилотного исследования, представленные в обработанном виде для дальнейшего анализа.
lingvotest_analysis — результаты статистического анализа.
lingvotest_Rscript — скрипт для проведения анализа в R.

zmanovsky_lingvotest_coursework — текст курсовой работы.
